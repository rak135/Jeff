# Research Shared Rules

Shared epistemic and output rules for research prompts.
Not yet runtime-wired. Candidate content for future composition into STEP1 and STEP3.

## Candidate rules

- Use only the provided evidence.
- Do not invent facts, sources, or certainty.
- All citation keys must come from the ALLOWED_CITATION_KEYS list.
- UNCERTAINTIES is always required; use the sentinel bullet if none are identified.
- No markdown, no code fences, no extra prose.
